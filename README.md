## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/andrii-pozniak/tasks-front.git
   ```

2. Install NPM packages
   ```sh
   npm install
   ```

3. Start the project
   ```sh
   npm start
   ```
